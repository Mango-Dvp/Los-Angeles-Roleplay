Getting Started

Welcome to the Los Angeles Roleplay Developer Documentation. This guide will help you integrate with our platform and build amazing experiences for our community.

Prerequisites

Before you begin, make sure you have:

  • A Los Angeles Roleplay developer account
  • Basic knowledge of RESTful APIs
  • Your preferred programming environment set up

Quick Start

Follow these steps to make your first API call:

  1. Register your application in the developer portal
  2. Obtain your API credentials
  3. Make a test request to verify your setup
Example: Basic API Request 
// JavaScript example using fetch
const response = await fetch('https://api.los-angeles-roleplay.online/v1/users', {
    method: 'GET',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
});

const data = await response.json();
console.log(data);

Authentication

All API requests require authentication using API keys or OAuth 2.0.

API Keys

For server-to-server communication, use API keys in the Authorization header:

Authentication Header 
Authorization: Bearer YOUR_API_KEY

OAuth 2.0

For user-facing applications, implement OAuth 2.0 flow to obtain access tokens.

OAuth 2.0 Authorization URL 
https://auth.los-angeles-roleplay.online/oauth/authorize
  ?response_type=code
  &client_id=YOUR_CLIENT_ID
  &redirect_uri=YOUR_REDIRECT_URI
  &scope=user:read

API Reference

Complete reference for all available API endpoints.

Users Endpoints

Method Endpoint Description
GET /v1/users Retrieve list of users
GET /v1/users/{id} Retrieve specific user
POST /v1/users Create a new user
PUT /v1/users/{id} Update user information

Characters Endpoints

Method Endpoint Description
GET /v1/characters Retrieve user characters
POST /v1/characters Create a new character
PUT /v1/characters/{id} Update character data
DELETE /v1/characters/{id} Delete a character

SDK & Libraries

Official client libraries to simplify integration with our API.

JavaScript/Node.js

Install via npm:

npm install 
npm install @los-angeles-roleplay/sdk

Usage example:

JavaScript SDK Usage 
const { LARPSDK } = require('@los-angeles-roleplay/sdk');

const client = new LARPSDK({
    apiKey: 'YOUR_API_KEY'
});

// Get user data
const user = await client.users.get('user_id');
console.log(user);

Python

Install via pip:

pip install 
pip install larp-sdk

Code Examples

Practical examples for common use cases.

Creating a New Character

JavaScript Example 
const characterData = {
    name: 'John Doe',
    age: 28,
    backstory: 'Former LAPD officer turned private investigator.',
    skills: ['investigation', 'firearms', 'interrogation']
};

const response = await fetch('https://api.los-angeles-roleplay.online/v1/characters', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify(characterData)
});

if (response.ok) {
    const character = await response.json();
    console.log('Character created:', character);
} else {
    console.error('Error creating character:', response.status);
}

Retrieving User Data

Python Example 
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.los-angeles-roleplay.online/v1/users/12345',
    headers=headers
)

if response.status_code == 200:
    user_data = response.json()
    print(f"User: {user_data['name']}")
else:
    print(f"Error: {response.status_code}")
